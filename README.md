# Wishlist App

Aplikasi Wishlist berbasis frontend menggunakan HTML, CSS, dan JavaScript.
Aplikasi ini memungkinkan pengguna untuk mengelola daftar keinginan dengan
fitur CRUD (Create, Read, Update, Delete) secara lengkap.

## Fitur Utama
- Menambahkan item wishlist
- Mengedit nama item secara langsung (inline edit)
- Menandai item sebagai "sudah dibeli"
- Menghapus item dari daftar
- Statistik jumlah item dan item yang sudah dibeli
- Dark Mode & Light Mode
- Desain UI modern dengan background gradient dan card warna-warni
- Penyimpanan data menggunakan localStorage (data tidak hilang saat refresh)

## Teknologi yang Digunakan
- HTML5
- CSS3
- Vanilla JavaScript

## Cara Menjalankan
1. Clone repository ini atau download sebagai ZIP
2. Buka file `index.html` menggunakan browser
3. Aplikasi siap digunakan

## Catatan
Aplikasi ini sepenuhnya berjalan di sisi frontend tanpa backend dan tanpa
framework tambahan, sesuai dengan ketentuan studi kasus.
